# QuickKart

**Instant Delivery at Your Fingertips**

QuickKart is an online shopping application that allows users to purchase a variety of products easily and quickly. With instant delivery capabilities, QuickKart brings your daily essentials right to your doorstep, offering a seamless shopping experience with exclusive deals, quick checkout, and real-time order tracking.


## Available Routes

- `/` - Home page
- `/product` - Product listing page
- `/login` - User login
- `/register` - User registration
- `/forgot-password` - Password recovery
- `/about-us` - About page (coming soon)
- `/contact` - Contact page (coming soon)


## Backend file created and initialized
npm init -y to initialize the package.json file on the backend
npm install express mysql2 bcrypt jsonwebtoken cors dotenv (other required necessary libraries)
